# 智慧社区业主端微信小程序

## 项目简介

这是一个基于微信小程序的智慧社区业主端应用，提供了完整的社区服务功能，包括报修管理、缴费管理、通知公告、活动管理、社区论坛、留言反馈和AI客服等模块。

## 配置说明

### 后端接口配置

在 `utils/request.js` 中修改 `baseURL` 为你的后端服务地址：

```javascript
const request = axios.create({
  baseURL: 'http://localhost:8080/api',  // 修改为实际的后端地址
  timeout: 10000
})
```

### 小程序配置

在 `app.json` 中可以配置：
- 页面路径
- 窗口样式
- 底部导航栏
- 网络超时时间

## 注意事项

1. **图片资源**：项目中使用了占位图片路径，实际使用时需要替换为真实的图片资源
2. **接口地址**：需要将 `utils/request.js` 中的 `baseURL` 修改为实际的后端接口地址
3. **网络请求**：小程序需要配置合法域名，请在微信公众平台配置服务器域名
4. **用户认证**：小程序需要获取用户授权，根据实际需求配置

## 开发说明

### 本地开发

1. 安装微信开发者工具
2. 导入项目目录
3. 修改后端接口地址
4. 点击编译运行

### 目录说明

- `pages/`：所有页面文件，每个页面包含 `.wxml`、`.wxss`、`.js`、`.json` 四个文件
- `utils/`：工具类和API封装
- `images/`：图片资源文件
