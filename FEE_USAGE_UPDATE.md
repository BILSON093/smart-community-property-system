# 缴费功能更新说明

## 更新内容

### 1. 数据库变更
创建了 `bus_fee_settings` 表来存储费用单价设置:

- `property_fee`: 物业费单价(元/㎡)
- `water_fee`: 水费单价(元/吨)
- `electricity_fee`: 电费单价(元/度)
- `gas_fee`: 燃气费单价(元/立方米)

### 2. 前端更新
**pages/pay/pay.js:**
- 添加了 `feeSettings` 数据字段
- 添加了 `loadFeeSettings()` 方法来加载费用设置
- 添加了 `getFeeSetting(type)` 方法来根据费用类型获取单价和单位
- 修改了 `loadFees()` 方法,自动计算用量(金额/单价)

**pages/pay/pay.wxml:**
- 在费用卡片中添加了用量显示(仅当有用量数据时显示)
- 在支付弹窗中添加了用量显示

### 3. 样式更新
**pages/pay/pay.wxss:**
- 将"立即支付"按钮颜色改为深色系 (#304156)

## 使用方法

### 步骤1: 执行数据库脚本
```bash
mysql -u root -p12345678 wuye < database/add_fee_settings.sql
```

### 步骤2: 重启后端服务
后端服务会自动创建默认的费用设置:
- 物业费: 2.50 元/㎡
- 水费: 3.80 元/吨
- 电费: 0.60 元/度
- 燃气费: 2.20 元/立方米

### 步骤3: 使用小程序
- 用户进入缴费页面
- 系统自动加载费用设置
- 根据总金额和单价计算用量
- 显示格式: "XX 吨"、"XX 度"、"XX 立方米"

## 用量计算公式

```
用量 = 总金额 / 单价
保留2位小数
```

例如:
- 水费: ¥45.00 ÷ ¥3.80/吨 = 11.84 吨
- 电费: ¥120.00 ÷ ¥0.60/度 = 200.00 度
- 燃气费: ¥32.00 ÷ ¥2.20/立方米 = 14.55 立方米

## 注意事项

1. 只有水费、电费、燃气费才显示用量,其他费用类型(如物业费)不显示
2. 如果后端返回的单价为空或0,则不显示用量
3. 用量只读显示,不能在前端修改
4. 管理员可以在后台修改单价设置
